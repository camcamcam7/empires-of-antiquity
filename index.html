How to play (controls)
— Selection —
Left-click — select   Left-drag — box select
Shift-click — add to selection
⌘/Ctrl-click or double-click — all of same type
— Commands —
Right-click — move / attack / gather
⌘/Ctrl + Right-click — attack-move
— Camera —
WASD / arrows — pan   Scroll / pinch — zoom
Mouse to screen edge — pan   Space — center on city
— Groups & Cycling —
⌘/Ctrl + 1–9 — assign group   1–9 — recall
. — next idle citizen   Tab — cycle buildings
— Build / Train (when selected) —
H F L M R T C D — build   1–8 — train troops
Building selected + right-click — rally pin
(rally on resource = gather, on enemy = attack)
— System —
P — pause   Esc — cancel
